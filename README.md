## Ramp Up

install node if you haven't already
http://nodejs.org/

install grunt (via terminal)
http://gruntjs.com/getting-started
```
npm install -g grunt-cli
```

Setting up grunt
```
#set current directory
cd projectDirectory

#install packages
npm install
```

Using grunt
```
#Run Grunt and grunt watch
grunt && grunt watch
```

You're set!
